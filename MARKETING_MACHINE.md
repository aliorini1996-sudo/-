# 🚀 آلية التسويق المتكاملة — Field Sales Marketing Machine

آلة تسويق ذاتية التشغيل 24/7 تغطي **كل الدول العربية** وتحوّل الغرباء إلى مشتركين — **مجانية بالكامل** (تعمل على GitHub Actions + الطبقات المجانية).

## خريطة الآلية — 7 أركان

```
       ① جذب داخلي (Inbound)                    ② صيد خارجي (Outbound)
  SEO: 1123 صفحة ثلاثية اللغة              صيد Leads كل 30 دقيقة (19 دولة عربية)
  GEO: llms.txt + محركات الذكاء                 ↓ إثراء بريد/هاتف تلقائي
       ↓                                    ③ سلسلة بريد 3 لمسات كل ساعتين
③ نشر اجتماعي يومي                          تعريف → قيمة/إثبات → رسالة أخيرة
  X (مرتان) · فيسبوك · لينكدإن ·                ↓ زاوية مخصّصة لكل دولة
  تيليجرام (جديد)                           ④ تتبّع فتح/نقر → عميل ساخن 🔥
       ↓                                         ↓
  ⑤ بحث القروبات كل ساعة              ⑥ غرفة قيادة التسويق (لوحة المالك)
       ↓                                         ↓
              ⑦ التحويل: تجربة مجانية → اشتراك (WON)
```

## ما الذي يعمل تلقائياً (بعد التفعيل)

| الركن | الجدولة | الملف |
|---|---|---|
| صيد عملاء محتملين (OSM+Geoapify+TomTom+Serper+Gemini) | كل 30 دقيقة | `leads-autohunt.yml` |
| سلسلة البريد التسويقي (3 لمسات، بحد يومي) | كل ساعتين | `leads-autoemail.yml` |

| نشر تيليجرام (منشور + بطاقة) | 7:30م KSA | `telegram.yml` |
| نشر فيسبوك / لينكدإن | مجدولة | `facebook.yml` / `linkedin.yml` |
| صيانة SEO+GEO (خريطة الموقع + llms.txt + تدقيق) | يومياً | `seo-maintenance.yml` |

## الجديد في هذه النسخة

1. **سلسلة متابعة 3 لمسات** بدل الرسالة الواحدة — المتابعات ترفع الردود 2-3×:
   - لمسة 1: تعريف · لمسة 2 (بعد 4 أيام): قيمة وإثبات (تسريب 5-12% من الإيراد) · لمسة 3: رسالة وداع مهذّبة.
   - المتابعات لها **الأولوية** في كل دفعة، والقوالب قابلة للتخصيص من اللوحة.
2. **زاوية مخصّصة لكل دولة** تُحقن تلقائياً في كل بريد: ZATCA للسعودية، ETA لمصر، FTA للإمارات، JoFotara للأردن، فرنسية للمغرب العربي، «يعمل مع إنترنت ضعيف» للعراق واليمن... (19 دولة).
3. **تتبّع فتح ونقر** (بكسل + روابط متتبَّعة + UTM): من يفتح درجته ≥7، من **ينقر** يصبح **مؤهّلاً 🔥** بدرجة 9 ويخرج من السلسلة وتظهر متابعته فوراً — اتصل به أنت.
4. **إلغاء اشتراك حقيقي بنقرة** (صفحة تأكيد ثنائية اللغة) — يحمي سمعة النطاق ويستثنيه كل الإرسال للأبد.
5. **تغطية كل الدول العربية**: زر «🌍 كل الدول العربية» في نافذة الصيد يفعّل 19 دولة بالتناوب.
6. **قناة تيليجرام** — نشر يومي مجاني بالكامل (البطاقة بنفس هوية X).
7. **غرفة قيادة التسويق** أعلى لوحة العملاء المحتملين: حالة المحرّكات، معدلات الفتح/النقر، العملاء الساخنون للتواصل الفوري، والتفاعل حسب الدولة.

## التفعيل (مرة واحدة)

### أ) المحرّكات الخلفية — من لوحة المالك (fieldsa.net → العملاء المحتملون)
1. **صيد مستمر** → زر «🌍 كل الدول العربية» → فعّل «الصيد التلقائي 24/7».
2. **بريد تلقائي** → راجع اللمسات (3) والفاصل (4 أيام) → فعّل «البريد التلقائي 24/7».
3. **بحث القروبات** → فعّل.

### ب) الأسرار (GitHub → Settings → Secrets → Actions)
| السر | الركن | من أين |
|---|---|---|
| `GEMINI_API_KEY` | كل النصوص والكلمات (Gemini 2.5) | aistudio.google.com/apikey — مجاني على هذا الحجم |
| `GEMINI_MODEL` *(اختياري، Variable لا Secret)* | اختيار النموذج | افتراضي `gemini-2.5-flash`؛ للجودة القصوى `gemini-2.5-pro` |
| `BREVO_API_KEY` | البريد (300/يوم مجاناً) | brevo.com — على خادم Render |
| `TELEGRAM_BOT_TOKEN` | تيليجرام | @BotFather (مجاني) |
| `TELEGRAM_CHAT_ID` | تيليجرام | @اسم_قناتك (اجعل البوت مشرفاً) |
| `X_API_*` (4 أسرار) | X | مضبوطة سابقاً |
| `AUTO_HUNT_TOKEN` | حماية الجدولة | أي نص عشوائي (اختياري، على Render + GitHub) |

> ملاحظة: `PUBLIC_API_URL` (اختياري على Render) يحدّد عنوان API لروابط التتبّع — الافتراضي `https://api.fieldsa.net`.

### ج) النشر
ادفع الكود (Render للخلفية + Vercel للواجهة يُعاد نشرهما تلقائياً). **لا تغيير على مخطط قاعدة البيانات** — كل الجديد يعمل فوراً.

## دورة العمل اليومية للمالك (5 دقائق)
افتح «غرفة قيادة التسويق» → إن وُجد **عملاء ساخنون 🔥** اتصل/واتسب بهم اليوم نفسه (هؤلاء فتحوا ونقروا — أعلى احتمال اشتراك) → راقب معدل الفتح (الصحي 20-40%؛ إن هبط دون 10% خفّض الحد اليومي وتحقق من DKIM).

## التكلفة الإجمالية
GitHub Actions + OSM/Geoapify/TomTom + Brevo (300 بريد/يوم) + Telegram + X المجاني + **Gemini 2.5 Flash** (طبقة مجانية سخيّة تكفي هذا الحجم) = **0$**. للجودة القصوى بدّل النموذج إلى `gemini-2.5-pro` (قد يتجاوز الحصّة المجانية اليومية مع صيد كل 30 دقيقة).

> **الذكاء الاصطناعي = Gemini 2.5 (Google)**: مفتاح واحد `GEMINI_API_KEY` يشغّل كل الأركان — النشر (X/تيليجرام/فيسبوك/لينكدإن عبر `marketing/post.py:ask_gemini`) وصيد/تأهيل العملاء (الخادم عبر `backend/src/services/gemini.ts`). أضِف المفتاح في **GitHub Secrets** (للنشر) و**Render env** (للصيد/التأهيل). النموذج قابل للضبط بمتغيّر `GEMINI_MODEL` في الاثنين.
